Every tool is prefixed `tdf_` and returns compact JSON. Parameters shared across
tools (`year`, `stage`, `lang`, `limit`, `offset`) are documented in
[Conventions](/docs/concepts). The tables below list each tool's own parameters
and the upstream endpoint it proxies.

## Race structure

Reference data for an edition — mostly stable before and during the race.

| Tool | Parameters | Endpoint |
| --- | --- | --- |
| [`tdf_event`](/docs/tools/race-structure#tdf_event) | — | `/api/event` |
| [`tdf_stages`](/docs/tools/race-structure#tdf_stages) | `year`, `stage?`, `full?`, `raw?`, +page | `/api/stage-{year}` |
| [`tdf_teams`](/docs/tools/race-structure#tdf_teams) | `year`, +page | `/api/team-{year}` |
| [`tdf_riders`](/docs/tools/race-structure#tdf_riders) | `year`, `bib?`, `team?`, `full?`, `raw?`, +page | `/api/allCompetitors-{year}` |

## Rankings & results

| Tool | Parameters | Endpoint |
| --- | --- | --- |
| [`tdf_rankings`](/docs/tools/rankings#tdf_rankings) | `year`, `stage`, +page | `/api/rankingType-{year}-{stage}` |
| [`tdf_stage_results`](/docs/tools/rankings#tdf_stage_results) | `year`, `stage`, +page | `/api/rankingTypeArrival-{year}-{stage}` |
| [`tdf_jersey_standings`](/docs/tools/rankings#tdf_jersey_standings) | `year`, `stage`, +page | `/api/rankingTypeJerseys-{year}-{stage}` |
| [`tdf_time_trial_rankings`](/docs/tools/rankings#tdf_time_trial_rankings) | `year`, `stage`, +page | `/api/rankingTypeTrial-{year}-{stage}` |
| [`tdf_rankings_widget`](/docs/tools/rankings#tdf_rankings_widget) | `year`, `stage`, +page | `/api/rankingTypeWidget-{year}-{stage}` |
